# Veri Madenciliği Ödevi

## Telco Customer Churn Analizi

Bu projede Orange Data Mining kullanılarak müşteri kaybı (Churn) analizi gerçekleştirilmiştir.

### Kullanılan Yöntemler
- Veri Ön İşleme
- Select Columns
- Impute
- Logistic Regression
- Random Forest
- Test & Score
- ROC Analysis
- Confusion Matrix

### Sonuçlar

#### Logistic Regression
- CA: 0.8481
- AUC: 0.8072

#### Random Forest
- CA: 0.9826
- AUC: 0.9316
